## 🚀 How to Use

### **Access the System:**
1. Open http://localhost:8501
2. Choose your role (Placement Officer or Student)
3. Use the sidebar to navigate between pages
4. Click logout to switch roles

### **Placement Officer Workflow:**
1. **Dashboard** → View overview
2. **Upload Files** → Add JDs and resumes
3. **Evaluate** → Run batch evaluations
4. **View Results** → Check scores and rankings
5. **Manage Data** → Browse all uploads

### **Student Workflow:**
1. **Apply for Jobs** → Upload resume once
2. **Browse Jobs** → View available positions
3. **Apply** → Get instant evaluation
4. **My Applications** → Track all applications
